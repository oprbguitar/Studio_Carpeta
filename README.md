# Herramientas para el Fiscal / FCIS — Demo estático

Demo preliminar de Fiscal Case Intelligence System (FCIS), una interfaz estática para organizar teoría del caso, hechos, elementos del delito, evidencias, medios probatorios, hipótesis fiscal, puntos débiles y próxima opción.

## Página en GitHub Pages

Abrir la demo publicada:

https://oprbguitar.github.io/Studio_Carpeta/

## Qué incluye

- Aplicación de una sola página en HTML, CSS y JavaScript puro.
- Dashboard, módulos del caso, paneles de detalle, pestañas, tarjetas, badges, KPIs y modo recorrido.
- Datos ficticios para el caso MP-2024-01567 por Robo Agravado.
- Exportación simulada de texto para algunos módulos.
- Diseño responsive y compatible con GitHub Pages.

## Uso local

Abrir `index.html` directamente en el navegador.

También se puede ejecutar:

```bash
python -m http.server 8000
```

Y abrir:

```text
http://localhost:8000
```
